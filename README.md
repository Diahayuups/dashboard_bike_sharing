# 🚲 Bike Sharing Data Analysis Dashboard

Proyek ini merupakan submission untuk kelas **Belajar Fundamental Analisis Data (BFDA)** di Dicoding.  
Analisis dilakukan menggunakan dataset **Bike Sharing** untuk memahami pola penyewaan sepeda berdasarkan berbagai faktor seperti kondisi cuaca, jenis hari, dan waktu penggunaan.

---

## 📊 Dashboard

Dashboard interaktif dibuat menggunakan **Streamlit** untuk menampilkan visualisasi hasil analisis data.

🔗 **Link Dashboard:**  
https://lana-dashboard-bike-sharing.streamlit.app/

## 📈 Pertanyaan Bisnis

1. Bagaimana pengaruh kondisi cuaca terhadap jumlah penyewaan sepeda pada sistem bike sharing selama tahun 2011–2012?
2. Bagaimana perbedaan jumlah penyewaan sepeda antara hari kerja dan hari libur pada sistem bike sharing selama periode 2011–2012?
3. Pada jam berapa aktivitas penyewaan sepeda paling tinggi dalam satu hari berdasarkan data tahun 2011–2012?

pip install -r requirements.txt

3. Jalankan Streamlit
streamlit run dashboard/dashboard.py
